# Chrome Web Store Listing Copy

## Extension Name
Meteora Axiom Button

## Short Description
Adds a quick Axiom trading button on Meteora DLMM pool pages.

## Detailed Description
Meteora Axiom Button adds a lightweight trading shortcut directly on Meteora DLMM pages.

What it does:
- Detects Meteora DLMM pool context on supported pages.
- Adds a floating button near the pool trading area.
- Opens Axiom in a popup window for quick trading flow.
- Reuses the same popup window on repeated clicks (no duplicate windows).
- Handles Meteora SPA navigation and route changes.

Notes:
- Designed for Solana-related Meteora/Axiom workflows.
- Uses minimal permissions required for operation.
- Does not run a custom backend.
